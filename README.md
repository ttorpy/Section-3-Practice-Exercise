# Section III Practice Exercise

### Requirements: Make a mad libs style story game with the following
- Accept ten total inputs (at least one as an int, one as a double, and one as a string)
- Do math with at least one integer input
- Do math with at least one double input
- Use at least one input in a question for the user

My story has ten total inputs with three integers, one double, and six strings. It uses math with one integer variable and one double variable. I learned that I can use the lower case method to convert a capitalized word stored inside a string variable into a lower case word. I got more comfortable with using variables combined with text inside of print statements. 

I also became more comfortable with parsing string inputs as integers or doubles and doing math with varaibles inside of print statements. I also learned how to use empty new-line print statements to break the displayed text into different paragraphs in the console.
